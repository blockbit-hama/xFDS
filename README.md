# FDS (Fraud Detection System) Service

Rust & Warp 기반의 실시간 사기 탐지 시스템 서비스

## 📌 Features

1. **지정 시간대 외 출금 요청 검출**  
   (기본: 09:00~18:00, 사용자별 커스텀 설정 가능)
2. **국외 IP 출금 요청 검출**  
   (사용자 평균 접속 위치 대비 편차 발생 시)
3. **60분 내 3회 이상 반복 출금**
4. **1회 최대 출금 한도 초과**  
   (₩100,000,000 / $80,000)
5. **1일 5회 이상 출금 시도**
6. **3개월 미접속 후 출금 요청**
7. **5일간 일평균 $1,000 이상 출금 패턴**
8. **미등록 IP 출금 요청**
9. **가입 후 24시간 내 첫 출금 시도**
